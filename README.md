## First step 
Clone your repository
```bash
    git clone  https://github.com/noam1610/trainingGithub
```


## Step two
Go to your repository in the terminal
```bash
    cd bla/bla/bla
```

## Step three
Synchronize your repository
```bash
    git sync
```
